# 🌌 Sirigiri Anand Kumar — Interactive Developer Portfolio

An Awwwards-level futuristic developer portfolio acting as an interactive operating system workspace, featuring a glassmorphic dashboard, an embedded command palette, real-time GitHub REST integration, and a fully interactive CLI terminal emulator.

---

## 🚀 Key Features

- **📺 Interactive Command Terminal**: A fully functional in-browser terminal command interpreter. Try typing `help`, `repos`, `skills`, `contact`, or `theme`.
- **⌨️ Raycast Command Palette**: Navigate the page, switch themes, or launch repositories directly using a Command Palette (`Ctrl + K` or `Cmd + K`).
- **🔄 GitHub Real-Time Node Syncing**: Live integration with the GitHub API to dynamically pull recent commit streams, repository specifications, and stack distribution graphs.
- **🌗 Dual-Mode OS Theme**: Seamless light and dark mode toggles with customized CSS overrides for CONTRIBUTION heatmaps and SVG charts.
- **⚡ Preloader Sync Pipeline**: Immersive startup preloader simulating the initialization of command matrices and remote sync socket pipelines.

---

## 🛠️ Stack & Technologies

- **Core Logic**: HTML5, Vanilla JS (ES6+)
- **Styling & Theme Framework**: Tailwind CSS (via CDN) + Custom CSS variables (glassmorphism details, CSS neon glow effects, and responsive timeline flows)
- **Data Pipeline**: Native GitHub API endpoints
- **Local Dev Server**: Node.js & `http-server`

## 🖥️ Getting Started & Running Locally

Follow these instructions to run the portfolio on your local machine:

### 1. Clone the repository
```bash
git clone https://github.com/Anandsirigiri07/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```
The server will spin up, and you can view the application in your browser at:
`http://localhost:3000`

## 🕹️ CLI Terminal Directives

Launch the terminal shell and try out these directives:

| Command | Action |
|---|---|
| `help` | List all available console directives |
| `repos` | Output top public repositories with stars and language details |
| `skills` | Aggregate active language stack percentages |
| `contact` | Output professional email and social coordinates |
| `theme` | Toggle the active visual theme (Light/Dark mode) |
| `clear` | Wipe terminal input history |
